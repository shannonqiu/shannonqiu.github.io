CAUTION: Do notremove the USB flash drive from the printer USB port while it is being accessedby the printer. This can damage the files on the drive. When removing the USB flash drive, make sure the USB port light is not blinking.

NOTE: The printer does not support encrypted USB flash drives or USB flash drives using the NTFS file system.
